GitHub Import Test - AI UX Web Components

This page demonstrates importing and using web components directly from GitHub Pages.

My Button Component

This component is loaded directly from GitHub Pages using a module import.

Live Example:

Event received! The component is working correctly.

Implementation:

<!-- Import the component from GitHub Pages -->
<script type="module" src="https://yourusername.github.io/aiux-experiments/lib/components/my-button.js"></script>

<!-- Use the component in your HTML -->
<my-button label="Imported from GitHub"></my-button>

Interaction Example:

// Get reference to the component
const myButton = document.querySelector('my-button');

// Listen for custom events from the component
myButton.addEventListener('button-click', () => {
  console.log('Button was clicked!');
});

// Change properties programmatically
myButton.setAttribute('label', 'New Label');

Integration Notes

  • Replace yourusername in the import URL with your actual GitHub username
  • Ensure your repository is published to GitHub Pages
  • For TypeScript projects, the component's type definitions are available in the dist directory
  • For production use, consider specifying a version in the URL (e.g., @v1.0.0)
